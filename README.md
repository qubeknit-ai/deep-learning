# 🧠 Deep Learning Repository

Welcome to this repository dedicated to all things deep learning! 🚀 This project is structured to keep all labs, notebooks, and datasets organized for easy access and experimentation.

> "this is deep lerning shit"

---

## 📂 Project Structure

This repository is organized into three main directories, ensuring everything has its place:

- **[`labs/`](./labs/)**: This directory houses all dedicated lab exercises and practical implementations.
- **[`notebooks/`](./notebooks/)**: Contains Jupyter Notebooks used for interactive development, visualization, and experimentation.
- **[`data/`](./data/)**: All datasets used throughout the project are stored here for consistency across labs and notebooks.

---

## 🛠️ Technology Stack

This project leverages some of the most powerful tools in the deep learning ecosystem:

- **TensorFlow**: For building and training neural networks.
- **Scikit-Learn**: For preprocessing, metrics, and traditional machine learning.
- **Pandas**: For robust data manipulation and analysis.
- **Matplotlib**: For high-quality data visualization and plotting.

---

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/qubeknit-ai/deep-learning.git
   ```
2. **Install dependencies**:
   Make sure you have your environment set up. If you're using `uv`, simply run:
   ```bash
   uv sync
   ```
3. **Explore!**: Check out the `labs/` folder for practical sessions or `notebooks/` for a deeper dive into the logic.
